## 브랜드 아이덴티티

- 브랜드명: **B.L.A.C.K (블랙)**
- 타겟: 10~20대(스트릿)
- 톤앤매너: 사이버펑크, 네온, 우천의 서울, 시네마틱
- USP: 힙한 패션템, 무광 블랙 디자인, 깨끗한 향기의 반전
- Main Copy : **뿌리는 게 아니다, 블랙을 입는 거다**
- Sub Copy :
  Sub 1 (디자인/태도): "엄마가 사둔 투명한 탈취제는 그만. 내 가방 속엔 시크한 무광 블랙."
  Sub 2 (향기/반전): "가장 다크한 블랙에서 터져 나오는, 갓 빤 티셔츠의 뽀송한 반전."
  Sub 3 (상황/기능): "땀 냄새, 피시방 냄새? 지우는 걸 넘어 너만의 무드로 덮어버려."
  Sub 4 (마무리): "진짜 멋은 보이지 않는 향기에서 결정되니까."


## 스토리보드
스토리보드(기획) 문서 (PDF 1개)

포함 항목: 브랜드 아이덴티티, 캠페인 목표/핵심 메시지, 씬 구성(씬별 필수 필드), 사용 도구 목록

씬별 필수 필드(모든 씬 공통):
씬 번호 / 씬 길이(초)
목표 메시지(한 문장)
화면 구성(구도/피사체/배경/텍스트 유무)
내레이션(또는 화면 카피)
사용 도구(이미지/비디오/오디오)와 사용 목적
입력 프롬프트(원문) + 출력 결과 요약(한 줄)
생성 결과 파일명 또는 링크 표기(파일명 규칙 자유, 단 일관성 유지)
추가 권장: 최소 1개 씬에 대해 “프롬프트 수정 전/후”와 수정 이유를 기록

기능 요구사항-------------------------------
1.기획(브랜드/캠페인) 정의
브랜드(가상 또는 실제) 1개를 선정하고 타겟/톤앤매너/차별점(USP)을 포함한 브랜드 아이덴티티를 정의한다.
광고의 목적(인지/전환/리텐션 등)과 핵심 메시지 1개를 한 문장으로 명확히 정의한다.

2.스토리보드 문서화
영상 제작 전 씬 단위로 스토리보드를 작성하고 씬별 필수 필드를 누락 없이 채운다.
최소 1개 씬에서 프롬프트를 "수정 전/후"로 비교하고 무엇을 왜 바꿨는지(의도/결과)를 기록한다.

3.멀티모달 생성 도구 사용
이미지 생성 AI 1종 이상, 비디오 생성(또는 변환) AI 1종 이상, 오디오 생성 AI 1종 이상을 사용한다.
도구는 자유롭게 선택하되 각 도구를 "왜 그 목적에 선택했는지"를 스토리보드에 간단히 남긴다.

4.영상 스펙(길이/구성)
최종 영상은 10초 이내로 제작한다.
영상에는 AI로 생성된 시각 요소(이미지/영상)와 청각 요소(음성/효과음/배경음악 중 1개 이상)가 반드시 포함되어야 한다.

5.단순 컷 나열이 아니라 최소한 다음 중 하나를 만족하는 스토리라인을 구성한다.
기승전결(문제 제시 → 전환 → 해결/제안) 구조
또는 명확한 메시지 전달("누구에게 어떤 가치를 주는지")이 드러나는 구조
영상의 마지막 3~5초 구간에는 브랜드/제품을 인지할 수 있는 장치(로고, 브랜드명, 슬로건, CTA 중 1개 이상)를 포함한다.
