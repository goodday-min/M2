## 브랜드 아이덴티티

#### 타겟: 10대 남학생(스타일과 스트릿 패션에 관심이 많은 그룹)

- 브랜드명 : **B.L.A.C.K (블랙)**
  (역발상 기획: 시크한 무광 블랙 용기 + 깨끗한 반전 향기)
- 향기 라인업: Black Cotton, Midnight Aqua, Urban Wood
- USP: 힙한 패션템, 무광 블랙 디자인, 깨끗한 향기의 반전
       단순한 생활용품(탈취제)이 아닌, 힙한 패션 아이템으로 포지셔닝
- 브랜드 컨셉 :  "가장 시크한 블랙, 가장 깨끗한 향기" (무광 블랙 용기 + 깨끗한 반전 향기) 
- 톤앤매너 :
  말투: 단호하고, 자신감 넘치며, 약간은 무심한 듯 시크한 말투
  느낌: 무신사 화보에 나오는 힙한 모델이나, 10대들이 동경하는 트렌디한 래퍼(형)가 툭툭 던지듯 조언해 주는 느낌
       "너 냄새나니까 씻어라"가 아니라, **"너 옷 잘 입었네? 근데 향기까지 챙겨야 진짜 멋있는 거야"**라고 쿨하게 알려주는 톤

- Main Copy : **뿌리는 게 아니다, 블랙을 입는 거다**
- Sub Copy :
  Sub 1 (디자인/태도): "엄마가 사둔 투명한 탈취제는 그만. 내 가방 속엔 시크한 무광 블랙."<br>
  Sub 2 (향기/반전): "가장 다크한 블랙에서 터져 나오는, 갓 빤 티셔츠의 뽀송한 반전."<br>
  Sub 3 (상황/기능): "땀 냄새, 피시방 냄새? 지우는 걸 넘어 너만의 무드로 덮어버려."<br>
  Sub 4 (마무리): "진짜 멋은 보이지 않는 향기에서 결정되니까."


## 스토리보드



포함 항목: 브랜드 아이덴티티, 캠페인 목표/핵심 메시지, 씬 구성(씬별 필수 필드), 사용 도구 목록

씬별 필수 필드(모든 씬 공통):
씬 번호 / 씬 길이(초)
목표 메시지(한 문장)
화면 구성(구도/피사체/배경/텍스트 유무)
내레이션(또는 화면 카피)
사용 도구(이미지/비디오/오디오)와 사용 목적
입력 프롬프트(원문) + 출력 결과 요약(한 줄)
생성 결과 파일명 또는 링크 표기(파일명 규칙 자유, 단 일관성 유지)
추가 권장: 최소 1개 씬에 대해 “프롬프트 수정 전/후”와 수정 이유를 기록




기능 요구사항-------------------------------
1.기획(브랜드/캠페인) 정의
브랜드(가상 또는 실제) 1개를 선정하고 타겟/톤앤매너/차별점(USP)을 포함한 브랜드 아이덴티티를 정의한다.
광고의 목적(인지/전환/리텐션 등)과 핵심 메시지 1개를 한 문장으로 명확히 정의한다.
------------------------------------------------------------------------------------
브랜드명 : BLACK
타켓 : 10대 남학생
톤앤매너: 사이버펑크, 네온, 우천의 서울, 시네마틱
- 
💡 "엄마가 사다 놓은 생활용품이 아니라, 내 가방에서 당당하게 꺼내는 '힙한 패션템'이다."

기존 페브리즈 같은 제품들과 비교해서 'BLACK'만이 가지는 차별점(USP)을 3가지로 쪼개서 설명해 드릴게요.

🖤 BLACK의 3가지 USP (차별화 포인트)
1. 시각적 차별화: "꺼낼 때 부끄럽지 않은 무광 블랙 디자인"

기존: 알록달록하고 투명한 플라스틱 통 (누가 봐도 '나 냄새나서 탈취제 뿌려요' 광고하는 느낌)
BLACK: 무신사에서 산 향수나 스트릿 브랜드 굿즈처럼 보이는 시크한 무광 블랙 용기. 친구들 앞에서 가방에서 꺼낼 때 오히려 멋있어 보입니다.
2. 후각적 반전: "블랙의 시크함 속에 숨겨진 갓 빤 티셔츠 향기"

기존: 뻔한 꽃향기, 인위적인 레몬향.
BLACK: 겉모습은 어둡고 강렬하지만, 뿌리는 순간 '블랙 코튼(Black Cotton)'의 뽀송하고 깨끗한 비누향이 퍼지는 반전 매력. (이 반전이 10대들에게 아주 매력적으로 다가갑니다.)
3. 감성적 가치: "냄새를 지우는 게 아니라, 핏(Fit)을 완성하는 것"

기존: "너 땀 냄새 나니까 이거 뿌려." (마이너스를 0으로 만드는 기능)
BLACK: "진짜 멋있는 애들은 향기마저 시크하게 관리해." (0을 플러스로 만드는 스타일링)


1. 광고 콘셉트 (Ad Concept)
한 줄 콘셉트: "보이지 않는 핏(Fit)의 완성, BLACK"
핵심 메시지: 냄새를 없애기 위해 마지못해 뿌리는 '탈취제'가 아니라, 외출 전 내 스타일을 완성하기 위해 마지막으로 걸치는 '패션 아이템'이다.
브리프 근거(Why?): 10대 남학생들은 옷, 신발, 머리 스타일 등 '눈에 보이는 멋'에는 관심이 많지만 향기 관리는 서툴거나 엄마가 사둔 마트용 탈취제에 의존합니다. 따라서 탈취제를 **'스타일링의 연장선(핏)'**으로 재정의하여, 가방에서 꺼내 뿌리는 행위 자체가 힙해 보이도록 유도하기 위함입니다.
2. 메인 카피 (Main Copy)
"뿌리는 게 아니다, 블랙을 입는 거다."

3. 서브 카피 (Sub Copy)
메인 카피를 뒷받침하며 제품의 매력(USP)을 구체적으로 찔러주는 문구들입니다.

Sub 1 (디자인/태도): "엄마가 사둔 투명한 탈취제는 그만. 내 가방 속엔 시크한 무광 블랙."
Sub 2 (향기/반전): "가장 다크한 블랙에서 터져 나오는, 갓 빤 티셔츠의 뽀송한 반전."
Sub 3 (상황/기능): "땀 냄새, 피시방 냄새? 지우는 걸 넘어 너만의 무드로 덮어버려."
Sub 4 (마무리): "진짜 멋은 보이지 않는 향기에서 결정되니까."

4. 카피 톤 설명 (Tone & Manner)
말투: 친절하고 다정한 존댓말(해요체)이 아닙니다. 단호하고, 자신감 넘치며, 약간은 무심한 듯 시크한 반말투를 사용합니다.
느낌: 무신사 화보에 나오는 힙한 모델이나, 10대들이 동경하는 트렌디한 래퍼(형)가 툭툭 던지듯 조언해 주는 느낌입니다. "너 냄새나니까 씻어라"가 아니라, **"너 옷 잘 입었네? 근데 향기까지 챙겨야 진짜 멋있는 거야"**라고 쿨하게 알려주는 톤입니다.






















2.스토리보드 문서화
영상 제작 전 씬 단위로 스토리보드를 작성하고 씬별 필수 필드를 누락 없이 채운다.
최소 1개 씬에서 프롬프트를 "수정 전/후"로 비교하고 무엇을 왜 바꿨는지(의도/결과)를 기록한다.

3.멀티모달 생성 도구 사용
이미지 생성 AI 1종 이상, 비디오 생성(또는 변환) AI 1종 이상, 오디오 생성 AI 1종 이상을 사용한다.
도구는 자유롭게 선택하되 각 도구를 "왜 그 목적에 선택했는지"를 스토리보드에 간단히 남긴다.

4.영상 스펙(길이/구성)
최종 영상은 10초 이내로 제작한다.
영상에는 AI로 생성된 시각 요소(이미지/영상)와 청각 요소(음성/효과음/배경음악 중 1개 이상)가 반드시 포함되어야 한다.

5.단순 컷 나열이 아니라 최소한 다음 중 하나를 만족하는 스토리라인을 구성한다.
기승전결(문제 제시 → 전환 → 해결/제안) 구조
또는 명확한 메시지 전달("누구에게 어떤 가치를 주는지")이 드러나는 구조
영상의 마지막 3~5초 구간에는 브랜드/제품을 인지할 수 있는 장치(로고, 브랜드명, 슬로건, CTA 중 1개 이상)를 포함한다.
