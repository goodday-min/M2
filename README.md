# 브랜드 아이덴티티

- 브랜드명: NeoSeoul
- 타겟: 10~20대(트렌디/스트릿)
- 톤앤매너: 사이버펑크, 네온, 우천의 서울, 시네마틱
- USP: “서울 감성 + 미래적 실루엣”을 결합한 한정 드롭
- 핵심 메시지(한 문장): “오늘 밤, 너의 서울을 입어라.”

스토리보드(씬 1 예시)

## 씬 1 (Intro)

- 씬 길이(초): 6
- 목표 메시지: “브랜드의 세계관(서울-네온-비)을 한 컷에 각인”
- 화면 구성: 네온사인이 반사되는 젖은 도로 / 인물 실루엣 / 로고는 없음
- 내레이션(또는 카피): “서울의 밤은, 다시 태어난다.”
- 사용 도구 및 목적:
    - 이미지 생성: Midjourney (키 비주얼 생성)
    - 비디오 변환: Runway (이미지를 짧은 모션 컷으로 변환)
    - 오디오: Suno (로파이 비트 + 빗소리)
- 입력 프롬프트(예시): cyberpunk neon Seoul night street, rain reflections, cinematic lighting, 8k, shallow depth of field
- 출력 결과 요약: 네온/우천/도시 질감이 강한 키비주얼 확보
- 결과 파일명(예시): scene01_keyvisual.png / scene01_motion.mp4 / scene01_bgm.wav
